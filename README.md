# FileDrop Advanced 🚀

A premium, modern Peer-to-Peer (P2P) file and text sharing application over the same Wi-Fi network.  
Built with vanilla WebRTC, PeerJS, Vite, and Cloudflare Workers for blazing-fast local transfers.

**👨‍💻 Developed by:** Md Shohag Islam  

## ✨ Advanced Features
- **Premium UI/UX:** Stunning Glassmorphism design with animated background mesh gradients.
- **Dark & Light Mode:** Seamlessly switch between themes with a smooth toggle.
- **Multi-File & Text Transfer:** Share multiple files at once or send text/links directly via WebRTC Data Channels.
- **Drag & Drop:** Easily drag files anywhere on the screen to instantly start sending.
- **No Server Storage:** Pure WebRTC P2P connection. Your files never touch a cloud server, ensuring 100% privacy.
